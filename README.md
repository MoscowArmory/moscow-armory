Moscow Armory Vocals Player is an open-source mobile application that gives singers and directors unprecedented control over individual voice parts in choral and sacred music recordings.

Perfect for learning parts, preparing rehearsals, studying sacred vocal traditions, or even live sound mixing.

### Multi-Track Audio Playback

- Real-time volume adjustment per channel
- Solo / Mute any combination of parts
- Create custom mixes for practice

### Professional USB Audio Interface Support (iPad)
- Dedicated landscape mixing console interface
- 4-channel audio output routing
- Piano routing to any voice channel(s)
